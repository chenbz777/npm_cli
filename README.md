## 安装

```
npm i @chenbz/cli -g
```



## 使用

```
# 打开终端输入 chenbz 即可使用
chenbz
```



## 介绍

使用命令工具来提高日常使用效率



## 功能

```
.
├── git
│   ├── 上推代码
│   ├── 拉取代码
│   ├── 提交代码到[本地]仓库
│   ├── 分支管理
│   ├── 克隆仓库
│   ├── 修改远程地址
├── npm
│   ├── 登录
│   ├── 退出登录
│   ├── 初次发布
│   ├── 发布
│   ├── 更新补丁号
│   ├── 更新次版本号
│   ├── 更新主版本号
```
